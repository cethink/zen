Fedora 21 always boots into Emergency Mode

2015-04-11
Fedora
约 282 字 预计阅读 1 分钟

文章目录

从 Fedora 20 升级到 21 之后，总是在启动时进入 Emergency Mode，每次都要按 Ctrl+D 才能进入系统，很是困扰。后来看提示输入命令journalctl -xb查看启动日志，发现有红色错误Fedora contains a file system with errors, check forced.，应该是磁盘的问题。

按照下面链接的指示，使用Solution#3

http://foobaring.blogspot.dk/2014/01/howto-run-fsck-on-emergency-mode-on.html

我用的是Ubuntu的LiveCD，进入系统后打开终端，输入sudo su进入root权限。

输入fsck /dev/sdaX命令进行磁盘检查，/dev/sdaX是/根路径所在分区，自动修复了一个错误，再进入系统时就不会进入 Emergency Mode 了。

突然想起升级后用 Disk Genius 修复了Windows的一个分区错误，看来都是分区的问题。
