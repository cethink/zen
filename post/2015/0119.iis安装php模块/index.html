IIS安装PHP模块

IIS 8.5 + PHP 5.6.5

PHP下载

选择NTS_VC11版，安装依赖Visual C++ Redistributable for Visual Studio 2012 Update 4。

将下载文件解压到C盘根目录，如C:\php。复制文件php.ini-development并改名为php.ini。

修改php.ini配置

; 拓展目录
extension_dir = "C:\php\ext"

; http://php.net/manual/en/timezones.asia.php
date.timezone = "Asia/Shanghai"

; 启用拓展
extension=php_gd2.dll
extension=php_mbstring.dll
extension=php_mysql.dll
extension=php_pdo_mysql.dll

配置IIS支持PHP

在Internet信息服务(IIS)管理器中，选择“处理程序映射”双击，最右边选择“添加模块映射”。 请求路径：*.php 模块：FastCgiModule 可执行文件：C:\php\php-cgi.exe 名称：PHP 确定，完成。

添加默认文档：default.php和index.php。
