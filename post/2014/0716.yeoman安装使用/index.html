文章目录

        • 命令

OS: Fedora 20

参考

http://yeoman.io/learning/index.html

命令

sudo npm install -g yo bower grunt-cli gulp
sudo npm install -g generator-webapp

mkdir yeoman
cd yeoman
yo webapp
bower install & npm install

grunt serve
